REQUIREMENTS:
  - Windows 10/11
  - Python 3.10 or higher (https://www.python.org/downloads/)
  - During Python install, CHECK "Add Python to PATH"

FIRST TIME SETUP:
  1. Install Python from https://www.python.org/downloads/
  2. Double-click "INSTALL.bat" (one time only)
  3. Wait for setup to complete

LAUNCH APPLICATION:
  Double-click "DWT Management System.bat"

DEFAULT LOGIN:
  Username: admin
  Password: admin123
  (Change password after first login from User Management)
